# Dimensionality Reduction Notebook

## Objective
Implement and compare PCA and t-SNE for dimensionality reduction using the Iris dataset.

## Steps Performed

1. Imported required libraries.
2. Loaded the Iris dataset.
3. Standardized the dataset.
4. Applied Principal Component Analysis (PCA).
5. Applied t-Distributed Stochastic Neighbor Embedding (t-SNE).
6. Visualized both results using scatter plots.
7. Compared the performance and visualization quality.

## Libraries Used
- pandas
- numpy
- matplotlib
- scikit-learn

## Conclusion
PCA reduced the dimensionality efficiently while preserving variance. t-SNE produced better visual cluster separation and is more suitable for visualization tasks.

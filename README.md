# Week 4 - Task 3: PCA & t-SNE Dimensionality Reduction

## Objective
This project demonstrates dimensionality reduction using Principal Component Analysis (PCA) and t-Distributed Stochastic Neighbor Embedding (t-SNE) on the Iris dataset. The objective is to reduce feature dimensions, visualize the data, and compare both techniques.

## Dataset
- Dataset: Iris Dataset
- Source: https://archive.ics.uci.edu/ml/datasets/iris

## Workflow
1. Load the dataset
2. Standardize the features
3. Apply PCA
4. Apply t-SNE
5. Visualize reduced dimensions
6. Compare PCA and t-SNE
7. Document observations

- Scikit-learn
